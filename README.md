# Reptile Tracker

## Get Started

### Install the dependencies

```bash
yarn
```

### Migrate Data

```bash
yarn migrate
```

## Development

### Start the server by running:

```bash
yarn dev
```

## Production

### Build the project by running

```bash
yarn build
```
